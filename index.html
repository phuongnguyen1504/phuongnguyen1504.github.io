Schedule Board – Chiba University

Professor Schedule Management · Client: Chiba University (Japan) · Toppa Solution

Web app for professors at Chiba University to manage their schedules — create meetings, book meeting rooms, and attach documents to calendar events. Used exclusively by the university's academic staff.

Key responsibilities:

  • Maintained and extended the system with a drag & drop time schedule feature, allowing professors to reschedule meetings intuitively
  • Integrated email notification to automatically remind all relevant participants when a meeting is scheduled or updated
  • Built new APIs and reworked UI components to match an updated Figma design
  • Implemented file attachment functionality for meeting documents

Technical challenge — Single-session login via WebSocket: The system required that each account could only be logged in on one device at a time. During implementation, three edge cases surfaced: (1) dropped connections where the server hadn't yet received a disconnect event, (2) race conditions when multiple devices logged in nearly simultaneously, and (3) users opening multiple browser tabs.

Resolved by: adding a heartbeat mechanism to detect dead connections, implementing an atomic session-update per userId to eliminate race conditions, and introducing a deviceId to distinguish sessions on the same device. Added a server-side API session check as a fallback to force logout even when the WebSocket event was missed.
